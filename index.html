HTML

  • The head element contains information about the webpage
  • The body element represents the visible contant shown to the user
  • Abbreviations
    • h1-h6: header
    • p: paragraph
    • ul: unordered list
    • ol: ordered list
    • li: list items
    • br: break
    • rel: relationship
    • href: hypertext reference

CSS

  • margin: space around the outside of an element
  • padding: space around the inside of an element
  • , : allows declarations to affect both selectors
  • .name: class selector

Git

  • git status: checks what branch we are on
  • git checkout branch-name: switches branch
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: commits all files in the working branch
  • git commit -m "": assosciates a message with our commit
  • git pull origin main: pulls base branch
  • git push origin branch-name: pushes all changes

JavaScript

  • const: create variables that can't be reassigned a value
  • let: variables that can be reassigned a value over time
  • functions: a block of code that performs a specific tasks
    • functions must be called in our code in order to execute them
